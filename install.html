Schritt 2: Manuelle Installation

Falls die automatische Installation nicht funktioniert, führe diese SQL-Befehle manuell aus:

mysql -u root -p
CREATE DATABASE phcard;
USE phcard;
SOURCE database.sql;

Schritt 3: Konfiguration

Stelle sicher, dass die Datei config.php die richtigen Zugangsdaten enthält:

define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', 'dein_passwort');
define('DB_NAME', 'phcard');
